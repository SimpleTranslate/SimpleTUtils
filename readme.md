# SimpleT Salesforce Knowledge Resources

This repository contains all the resources, code, and utilities required to set up and manage SimpleT's components. It is designed to help admins, developers, and users quickly deploy and configure the necessary features.

---

## Repository Contents

- **Salesforce Services & Triggers** – apex classes required for setting up some of our features
- **Utility Scripts & Helpers** – Additional tools and scripts for managing data
- **Documentation** – Setup guides

---

## Getting Started

1. Browse the folders to find the component you want to implement.  
2. Follow the included setup guides for each component.  
3. Copy and paste code into your Salesforce org using your preferred deployment method.
4. Review permissions and configuration instructions to ensure proper functionality.
